# PROJET_CREFER

Scaffold initial — Vue 3 + Vite + Tailwind CSS

Prérequis
- Node.js 18+ (ou récent)
- npm (ou pnpm/yarn)

Installer les dépendances
(PowerShell)

npm install

Démarrer le serveur de dev

npm run dev

Build de production

npm run build

Notes
- J'ai inclus `tailwind.config.cjs` et `postcss.config.cjs` ainsi que `src/assets/index.css` contenant les directives Tailwind.
- Après `npm install`, Vite servira l'app et Tailwind générera les classes à partir des fichiers listés dans `tailwind.config.cjs`.
